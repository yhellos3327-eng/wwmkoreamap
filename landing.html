기술 스택

최신 웹 기술로 구현된 고성능 지도

수천 개의 마커를 부드럽게 렌더링하고, 실시간 동기화를 지원합니다.

PixiJS

GPU 가속 렌더링

WebGL 기반 하드웨어 가속으로 4,000개 이상의 마커를 60fps로 부드럽게 렌더링합니다. CPU 렌더링 대비 10배 이상의 성능 향상.

Leaflet + 커스텀 타일

인터랙티브 지도 엔진

고해상도 맵을 타일 단위로 최적화하여, 언제 어디서나 빠르고 쾌적한 로딩을 제공합니다.

Firebase

실시간 클라우드 백엔드

Firestore로 실시간 댓글 동기화, Authentication으로 카카오/구글 OAuth 지원. 서버리스 아키텍처로 무한 확장 가능.

Supercluster

마커 클러스터링

Mapbox 오픈소스 알고리즘으로 근접 마커 자동 그룹화. 줌 레벨에 따라 동적으로 클러스터 분할/병합.

AI 번역 엔진

OpenAI / Gemini / Chrome AI

미번역 콘텐츠를 실시간 AI 번역. Chrome 내장 Translator API 우선 사용으로 비용 절감, 폴백으로 클라우드 AI 연동.

IndexedDB + 캐싱

오프라인 퍼스트

완료 체크, 즐겨찾기를 로컬에 즉시 저장 후 클라우드 동기화. 네트워크 없이도 데이터 유지. LRU 캐시로 메모리 최적화.
